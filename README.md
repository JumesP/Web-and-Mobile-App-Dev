# Web and Mobile Application Development Project
 
This is my project for my Web and Mobile Application assignment for University.

The requirements for the project was to create a complete website (Frontend and Backend) the following technologies. The application much include 3 different pages, connect to a database and be accessable via an online URL, so it must be hosted.

### The Website is accessable at the following link:
https://theonlinewebstore.azurewebsites.net/


<!--Within this project, i learnt how webpages connect to databases, present data and operate from a backend point of view, allowing me to further my understanding of how websites and the entire internet works!-->

## This project is created using:
- NodeJS
- ExpressJS
- Embedded JavaScript (EJS)
- JQuery
- Bootstrap
- MongoDB
- Mongoose (MongoDB connector)
- plus a variery of libs...

## Project Description:

This application contains 3 pages, A landing page, a storefront and a specific product page.

### Landing:

The Landing page presents a beautiful introduction to the website.

This was created via overlaying a grey box over an image of clothes, to provide a modern and premium design.

The Store Front features all the products sold by the store, as well as details including name, price and rating.

This was created using EJS to list all products in the database.

The Specific Product page outlines all the details of each product.
